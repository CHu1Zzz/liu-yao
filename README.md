# 六爻占卜

古典六爻占卜网站，包含硬币起卦、时间起卦、随机起卦，以及卦象解析、经典原文对照与互动学习。

## 功能

- **三种起卦方式**：硬币起卦、时间起卦、随机起卦
- **卦象解析**：六爻排列、世应定位、用神选取
- **经典原文**：《火珠林》《增删卜易》《卜筮正宗》精华对照
- **学习中心**：六爻基础知识，浅学/深学模式切换
- **互动练习**：十道精选练习题

## 技术栈

- React + Vite
- Tailwind CSS
- React Router

## 运行

```bash
npm install
npm run dev
```
